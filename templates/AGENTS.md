# Repository Instructions

Read this file before editing. It is the source of truth for agents in this repository.

## Setup and checks

- Install dependencies: `<command>`
- Run focused tests: `<command>`
- Run full checks before PR: `<command>`

## Workflow

- Keep changes small and tied to the request.
- Read `CONTEXT.md` for architecture and purpose when present.
- Read `STYLE.md` for project conventions when present.
- Use a generate/evaluate/repair loop for non-trivial work.
- Do not claim completion until the relevant check has actually run.

## Safety

- Do not commit secrets, local config, logs, generated credentials, or private data.
- Ask before destructive, credentialed, public, or expensive actions.

## Handoff

If work is interrupted, record branch, changed files, checks run, blockers, and next action in the repository's handoff location.
