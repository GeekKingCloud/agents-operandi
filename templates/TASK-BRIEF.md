# Task Brief

## Mode

implement / report-only / review / recovery / research.

## Target

Exact repo, path, file, branch, session, or artifact the work applies to.

## Source of truth

Exact branch, repo, raw source, or ticket that decides correctness. Installed copies, scratch dirs, and summaries are evidence only unless explicitly designated here.

## Boundary

Touch X, ignore Y, ask before Z.

## Autonomy

act / recommend / ask before mutation / stop at review / local-only / specifically authorized remote actions.

## Review gate

self-review / purposeful sub-agents with named roles / Roast / another required reviewer / explicitly unreviewed.

## Success check

Exact command, artifact, or state that proves the goal.

## Stop rule

Stop on material ambiguity, failed proof, long-running stall without new evidence, or a new authority requirement.
